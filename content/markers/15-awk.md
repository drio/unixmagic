[AWK](https://en.wikipedia.org/wiki/AWK) is a tiny programming language
aimed squarely at text: read input line by line, match patterns, run
actions, emit output. Great on its own, better inside a pipeline.

It was written at Bell Labs in the 1970s. The name comes from the three
authors' initials: **A**lfred Aho, Peter **W**einberger, and Brian
**K**ernighan — the same "k" as in [#5](#annotation-05-kandr).
