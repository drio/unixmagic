The [`man(1)`](https://man7.org/linux/man-pages/man1/man.1.html) command
(short for "manual") opens the reference pages that ship with the system:
commands, system calls, library functions, config files, devices. Before
the web, these pages *were* the documentation — and for a long time after,
they were still the fastest way to get an authoritative answer without
leaving the terminal.

The figure in the window is harder to pin down. He's holding a scythe,
which could fit: on Unix, a parent process "reaps" its children by
reading their exit status, letting the kernel clear them out of the
process table. Others read him as a [hacker](https://en.wikipedia.org/wiki/Hacker_culture)
in the older sense — the clever tinkerer, not the intruder.
